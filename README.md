## BandwagonHost 是做什么的

BandwagonHost 卖的是 VPS——Virtual Private Server，虚拟专用服务器。

可以把它理解成：你在远程机房租了一台一直开着的 Linux 服务器，你有完整的 root 权限，可以装软件、跑程序、建网站、部署各类服务。和国内云服务商的区别是：搬瓦工在境外，不需要 ICP 备案，机器用的是境外 IP。

它在国内用户里口碑不错，核心原因有两个：

1. **CN2 GIA 线路**——这是电信专门为跨境流量优化的精品路由，高峰期稳定，延迟低。对需要跑境外服务的人来说，这条线路比普通国际带宽强很多。
2. **支付宝可以直接付**——境外服务能直接支付宝付款的不多，搬瓦工算其中比较方便的。

## 搬瓦工官网英文界面：注册流程图解

官网全程英文，但逻辑其实很直。找准位置就不绕。

**1. 创建账号（Register）**

打开官网右上角，找 "Register" 按钮，进入注册页面。需要填的信息：

- **First Name / Last Name**：名字，可填拼音
- **Email Address**：邮箱，后续订单和工单通知都发这里
- **Password**：密码
- **Country**：选 China 或你实际所在地
- **State/Region**：省份

**注意**：填国家和省份要和你付款时的信息一致。搬瓦工有风控系统，IP 地址、注册信息、付款信息差异太大会被判断为欺诈订单拒绝付款。

填完点 Register，收到验证邮件激活账号就好了。

**2. 选套餐下单**

登录后进入 "VPS Hosting" 或直接点套餐链接。每个套餐页面能看到配置和价格，选好之后：

- 点 **Order** 进入购物车
- 可以选计费周期（月付、季付、年付），年付通常单价最低
- 有优惠码的话在 **Promotional Code** 框填入
- 确认金额，点 **Checkout**

**3. 付款**

支持方式：支付宝、PayPal、信用卡、银联。

选支付宝的话，会跳转到支付宝扫码页面，手机扫码付款，回到官网确认完成。VPS 实时开通，一般几分钟内就好了。

**4. 管理 VPS：KiwiVM 控制面板**

付款完成后，进 "My Services" 找到你的 VPS，点进去进入 KiwiVM 管理面板。在这里可以：

- 重装系统（支持 CentOS、Ubuntu、Debian、Rocky Linux 等 20+ 版本）
- 开机/关机/重启
- 查看流量统计
- 切换数据中心（部分套餐支持）
- 设置 SSH 密钥、RDNS

整个面板操作基本是中文用户也能靠图标和按钮名称摸清楚的。确实看不懂的，对照英文关键词搜一下就找到教程了。

## CN2 和 CN2 GIA 的区别：搞清楚再买

这个问题问的人很多，简单说清楚。

**CN2（GT）**：中国电信的商业线路之一，代号 AS4134。比普通国际带宽质量好，但高峰期会有拥堵，丢包率不稳定。

**CN2 GIA**：电信最高等级的精品直连线路，代号 AS4809。专线质量，高峰期保持稳定，丢包率低，延迟更可控。代价是贵——电信自己的批发价就已经很高了，最终到消费端价格自然也高。

搬瓦工目前主推的 CN2 GIA-E（E 代表 eCommerce，就是商用级别）走的是 DC6、DC9 机房，回程三网全走 CN2 GIA。日常实测从洛杉矶到国内，延迟大概在 130-160ms，高峰期没有明显波动。
